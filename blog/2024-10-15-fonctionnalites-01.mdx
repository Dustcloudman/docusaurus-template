# Présentation des Fonctionnalités du Modèle de Code Personnalisé

Dans cet article, je vous présente les principales fonctionnalités du modèle de code que j'ai personnalisé. Ce modèle est conçu pour être facilement utilisable et flexible, que vous soyez un développeur expérimenté ou un novice.

{/* truncate */}

## Fonctionnalités Déjà Ajoutées

:::success 3 Sidebars Auto-générées

Les **sidebars** sont **auto-générées**, et chaque sidebar pour les documents est indépendante, permettant une organisation claire et une navigation fluide entre les différentes catégories.

:::

### Documentation

- **3 sidebars** : Chaque sidebar est indépendante et parfaitement organisée pour une meilleure accessibilité des documents.
- **Pages index pour les catégories** : Chaque catégorie a sa propre page index, facilitant la navigation entre les sections.
- **Prise en charge du .mdx** avec **React** : Ajoutez facilement du contenu interactif à vos documents grâce au support du `.mdx`.
- **Codé en TypeScript** : Une configuration robuste et évolutive, idéale pour les projets complexes.
- **Fichier de configuration** simple et clair : Vous pouvez rapidement comprendre et modifier les paramètres essentiels.
- **Preset Classic** : Utilisation du preset Docusaurus classique pour une configuration rapide et efficace.

### Blog

- **3 blogs distincts** : Chaque blog a son propre espace et ses auteurs, indépendamment des autres blogs.
- **Organisation simple** : Les articles sont organisés selon la structure **YYYY/MM/DD-titre-du-post.mdx**, ce qui permet une gestion facile.
- **Accessible depuis GitHub** : Le modèle est prêt à l'emploi pour des personnes avec peu de connaissances en code. Il peut être utilisé directement depuis GitHub ou via le navigateur avec une simple configuration de l'URL du dépôt.

:::info Prêt à l'emploi

Ce modèle est conçu pour être **intuitif** et **facile à maintenir**, même pour des utilisateurs ayant peu de connaissances techniques.

:::

## Fonctionnalités à Venir

:::note Fonctionnalités en Développement

Certaines fonctionnalités sont en cours de développement pour rendre ce modèle encore plus puissant.

:::

### Internationalisation

- **i18n** : Ajout prochain de la fonctionnalité d'**internationalisation** pour permettre la gestion multilingue du site.

### Barre de Recherche Privée

- **Typesense** : Utilisation de la solution open source **Typesense** pour implémenter une **barre de recherche privée**, facilitant la recherche dans les documents et blogs hébergés dans des répertoires privés.

### Améliorations des Sidebars

- **Sidebar.ts** : Une configuration encore plus flexible sera disponible via le fichier `sidebar.ts` pour gérer différents types de sidebars selon vos besoins.

### Mise à jour du Code de Configuration

Des ajustements réguliers seront faits pour améliorer la structure et la maintenabilité du fichier de configuration, en s'appuyant sur les retours utilisateurs.
