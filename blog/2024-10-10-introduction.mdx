# Mon Parcours avec Docusaurus : Utilisation Privée et Personnalisée

J'ai choisi de me plonger dans le code de **Docusaurus** par moi-même, sans aucune compétence préalable en développement. Après quelques tentatives et déconvenues avec d'autres CMS, j'ai trouvé que Docusaurus pouvait être particulièrement utile dans certaines situations.

{/* truncate */}

:::note Utilisation Privée

J'aime **détourner** l'utilisation classique de Docusaurus en tant que documentation pour des solutions de type **Software as a Service (SaaS)**, et l'utiliser de manière privée.

:::

## Utilisation Personnelle (Accès Sécurisé par Login)

Voici comment j'utilise ce modèle personnalisé de Docusaurus pour des **blogs unipersonnels** :

- **Journal intime** : Suivi de mes réflexions personnelles au quotidien.
- **Journalisation des interactions** : Un espace pour suivre les interactions avec diverses personnes dans des situations variées.
- **Mémoire des évènements importants** : Conserver une trace des moments marquants de ma vie.
- **Journal de projets à long terme** : Rédiger des narratifs sur l'évolution de mes projets sur le long terme.
- **Organisation par tags et dates** : Chaque entrée est étiquetée et datée, facilitant la recherche future.
- **Recherche dans l'application** (fonctionnalité à venir) : Une recherche interne pour naviguer efficacement dans les entrées du blog.

:::success Blogs Distincts

J'ai choisi de mettre en place **3 blogs distincts** pour permettre une utilisation flexible et universelle de Docusaurus à titre individuel :

:::

1. **Privé**
2. **Professionnel**
3. **Couple**

Ces blogs sont nommés de manière **générique** dans le modèle, car chacun peut les personnaliser en fonction de ses propres besoins et préférences.

## Documentation Interpersonnelle (Accès Sécurisé par Login)

En complément des blogs, j'ai intégré un **module de documentation** accessible de manière **interpersonnelle**. Cela permet de documenter tout ce qui peut être nécessaire avec un **narratif structuré et facile à retrouver**.

:::info 3 Sidebars pour Organiser la Documentation

J'ai configuré le modèle avec **3 sidebars** pour couvrir les aspects **privés**, **professionnels**, et **de couple**, de manière bien organisée.

:::

Ces sidebars permettent de naviguer facilement entre les différentes sections de la documentation, offrant ainsi un support narratif structuré pour accompagner les blogs.

## Conclusion

Le travail sur ce projet est réalisé **"dans l'état"**, avec le temps que j'ai à disposition. 

Pour toute question ou retour d'expérience, **n'hésitez pas à me contacter**. Ce modèle est ouvert à toute adaptation personnelle !

:::caution Travail en Cours

Certaines fonctionnalités, comme la recherche dans l'application, sont en cours de développement.

:::
