## Sidebar config

I use simple autogenerated Sidebar



```bash sidebar.ts
import type {SidebarsConfig} from '@docusaurus/plugin-content-docs';

const sidebars: SidebarsConfig = {
sidebar01: [
  {
    type: 'autogenerated',
    dirName: 'sidebar01',
  },
],
sidebar02: [
  {
    type: 'autogenerated',
    dirName: 'sidebar02',
  },
],
};

export default sidebars;
```

### sidebar practical knowledge

```bash /docs directory structure
sidebar01 
├── 01-category01
│   ├── 06-category06 ## /docs/sidebar01/category01/category06/
│   │   │ ├── 02-category02 ## /docs/sidebar01/category01/category06/category02/
│   │   │        └── 01-topic1.mdx ## /docs/sidebar01/category01/category06/category02/topic1   
│   │   └── 01-topic1.mdx
│   │   ├── 03-topic3.mdx 
│   │   └── index.mdx (url from directory overtaken by managed site generator config)
│   └── 01-topic1.mdx
│   ├── 02-topic2.mdx
│   └── 03-topic3.mdx ... until 05 as number 6 is category, then you can keep going
├── 02-category02
│   ├──copy/paste category01 ## multi-sidebar ready
```
